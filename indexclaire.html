DFN

Représente le terme d'une définition, on l'utilise dans un paragraphe ou une liste de définitions.

Exemple :

Le SEO est un ensemble de techniques pour référencer un site Web

Paramètres d'affichage CSS par défaut :
La plupart des navigateurs vont afficher l'élément <p> avec les valeurs suivantes : 

 
                     dfn {
 			   font-style: italic;
			}
DEL

Utilisé pour indiquer qu'une partie de texte est supprimée, mais conservée pour en garder la trace, on la stylisera en CSS.

Exemple :

My favorite color is blue red!

Paramètres d'affichage CSS par défaut :
La plupart des navigateurs vont afficher l'élément <p> avec les valeurs suivantes : 

 
                      del {
    			   text-decoration: line-through;
			}
I

Utilisée dans un paragraphe pour indiquer qu'une partie du texte est différent du reste.

Exemple :

He named his car The lightning, because it was very fast.

Paramètres d'affichage CSS par défaut :
La plupart des navigateurs vont afficher l'élément <p> avec les valeurs suivantes : 

 
                       i {
    			  font-style: italic;
			}
INS

Pour insérer un nouveau texte dans un document révisé. On peut utiliser cette balise avec qui permet d'indiquer qu'une portion de texte n'est plus valable.

Exemple :

My favorite color is blue red!

Paramètres d'affichage CSS par défaut :
La plupart des navigateurs vont afficher l'élément <p> avec les valeurs suivantes : 

 
                       ins {
    			text-decoration: underline;
			}
